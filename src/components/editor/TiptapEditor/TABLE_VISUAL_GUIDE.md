# 📋 表格功能快速指南

## 🔍 为什么看不到表格操作？

**答案：** 表格操作菜单只在光标位于表格内时显示！

### ✅ 正确的使用方式：

```
步骤 1: 插入表格
┌─────────────────────────────────────┐
│  工具栏 → [📋 表格图标] → 插入表格  │
└─────────────────────────────────────┘

步骤 2: 点击表格内的单元格
┌─────────────┬─────────────┐
│  [点击这里]  │             │  ← 点击表格内的任意位置
├─────────────┼─────────────┤
│             │             │
└─────────────┴─────────────┘

步骤 3: 再次点击工具栏的表格图标
┌─────────────────────────────────────┐
│ 现在你会看到完整的菜单：            │
│ ├─ 插入表格 (2x2)                   │
│ ├─────────────────────────────────  │
│ ├─ 在上方添加行                     │
│ ├─ 在下方添加行                     │
│ ├─ 复制当前行 ✨                    │
│ ├─ 删除当前行                       │
│ ├─────────────────────────────────  │
│ ├─ 在左侧添加列                     │
│ ├─ 在右侧添加列                     │
│ ├─ 复制当前列 ✨                    │
│ ├─ 删除当前列                       │
│ ├─────────────────────────────────  │
│ ├─ 切换表头列                       │
│ ├─ 切换表头行                       │
│ ├─ 合并/拆分单元格                  │
│ └─ 删除表格                         │
└─────────────────────────────────────┘
```

### ❌ 错误的使用方式：

```
步骤 1: 光标在表格外的段落中
┌─────────────────────────────────────┐
│ 这里是一段文字 [光标在这里]         │  ← 光标不在表格内
└─────────────────────────────────────┘
         ↓
步骤 2: 点击表格图标
┌─────────────────────────────────────┐
│ 只能看到：                          │
│ └─ 插入表格 (2x2)                   │  ← 只显示这一个选项
└─────────────────────────────────────┘
```

## 🎯 完整示例操作流程

### 插入并操作表格的完整步骤：

1. **输入一些文字**
   ```
   这是一个示例文档
   ```

2. **插入表格**
   - 点击工具栏的 📋 图标
   - 选择"插入表格 (2x2)"
   - 会出现一个表格

3. **点击表格内的第一个单元格**
   - **重要！** 必须点击表格内部

4. **再次点击工具栏的 📋 图标**
   - 现在看到所有操作选项了！

5. **尝试一些操作**
   - 点击"在下方添加行" → 会添加一行
   - 点击"在右侧添加列" → 会添加一列
   - 点击"复制当前行" → 会复制当前行的内容

6. **调整列宽**
   - 移动鼠标到单元格右边缘
   - 看到蓝色竖线时拖拽

## 🐛 如果还是看不到...

### 调试步骤：

1. **确认表格已插入**
   - 应该能看到一个 2x2 的表格

2. **确认光标在表格内**
   - 点击表格中的单元格
   - 单元格应该有选中效果

3. **刷新浏览器**
   - 按 `Ctrl + R` 或 `F5`
   - 清除缓存后重试

4. **检查控制台**
   - 按 `F12` 打开开发者工具
   - 查看是否有错误信息
